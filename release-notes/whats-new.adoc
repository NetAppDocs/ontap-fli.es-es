:hardbreaks:

[role="lead"]
Conoce las nuevas funcionalidades disponibles en ONTAP para la migración de datos SAN.



[[ontap-9-19-1]]
== ONTAP 9.19.1

* A partir de ONTAP 9.19.1, ONTAP FLI es compatible con el protocolo iSCSI para la conectividad backend.




[[ontap-9-17-1]]
== ONTAP 9.17.1

* A partir de ONTAP 9.17.1, los sistemas ASA R2 son compatibles con la migración FLI.
+
Obtenga más información sobre link:https://docs.netapp.com/us-en/asa-r2/get-started/learn-about.html["Sistemas ASA R2"^] .
