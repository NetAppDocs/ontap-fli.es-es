== Post-migración - FLI en línea

Después de eliminar la importación del LUN, verifica si el LUN está configurado con `-space-reserve disabled` y `-space-allocation enabled`. Si no, modifícalo en consecuencia.

Para que la función SCSI UNMAP funcione correctamente, consulta la documentación del proveedor para verificar las versiones de software compatibles. Se requiere la combinación de Veritas VxVM + VxFS para activar esta función.

[cols="4"]
|===
| Sistema de archivos | Dispositivo de bloques | LVM (FS) | Sanboot 


| Paso 1: Volver a explorar el almacenamiento en todos los hosts HP-UX asignados | Paso 1: Vuelve a explorar el almacenamiento en todos los hosts HP-UX asignados. | Paso 1: Volver a explorar el almacenamiento en todos los hosts HP-UX asignados | Paso 1: Volver a explorar el almacenamiento en todos los hosts HP-UX asignados 
|===
.Pasos
. Identifica los dispositivos de almacenamiento.
+
[source, cli]
----
ioscan -fnC disk
insf -e
sanlun lun show all
sanlun lun show all -pv
sanlun lun show -wwpn
----
. Importar VG y poner en línea el VG.
+
[source, cli]
----
vgimport /dev/vg_name /dev/disk/disk_name
vgchange -a y /dev/vg_name
----
. Identifica el grupo de volúmenes y los sistemas de ficheros.
+
[source, cli]
----
vgdisplay -v /dev/vg_name
vgdisplay /dev/vg_name
lvdisplay /dev/vg_name/lv_name
----
. Monta el sistema de archivos.
+
[source, cli]
----
bdf
mount /dev/vg_name/vol_name /mount_point
----
